# DevSwitch

本上下文描述 DevSwitch 这个 Windows 原生桌面工具的产品语言。该工具帮助开发者管理并切换本机已安装的开发工具链版本。

## Product

**DevSwitch**:
面向 Windows 开发者的图形化开发工具链版本切换工具。
_Avoid_: EasySDK、SdkSwitch、环境变量编辑器

## Language

**SDK 切换工具**:
用于管理本机开发工具链版本，并让用户选择当前生效版本的桌面工具。
_Avoid_: 环境变量工具、下载器、包管理器

**开发工具链版本**:
某一种开发工具或运行时的具体安装版本，例如某个 JDK、Maven、Node.js 或 Go 版本。
_Avoid_: 软件包、依赖、插件

**全局切换工具**:
以整台 Windows 用户环境为作用范围切换当前生效开发工具链版本的工具，不按项目目录自动选择版本。
_Avoid_: 项目级切换、自动切换器、版本下载器

**首版 SDK 类型**:
第一版需要支持的开发工具链类别：JDK、Maven、Node.js 和 Go。
_Avoid_: 全语言支持、插件全覆盖、运行时合集

**本地导入**:
用户选择已经存在于本机的开发工具链安装目录，工具识别后登记为可切换版本。
_Avoid_: 安装、下载、扫描全盘

**SDK 根目录**:
某个开发工具链版本的安装根目录，工具通过根目录下的标志文件或子路径确认其类型和版本。
_Avoid_: bin 目录、命令目录、可执行文件目录

**在线下载**:
工具从配置的版本源获取开发工具链安装包，完成下载、校验、解压并登记为可切换版本。
_Avoid_: 导入、包管理、依赖安装

**外部 SDK**:
通过本地导入登记、实体文件仍保留在用户原始位置的开发工具链版本。
_Avoid_: 托管 SDK、复制版本、工具安装版本

**托管 SDK**:
由工具通过在线下载创建并保存在数据根目录内的开发工具链版本。
_Avoid_: 外部 SDK、系统安装版本、仅登记路径

**删除 SDK 记录**:
从工具的可切换版本列表中移除某个开发工具链版本；外部 SDK 只移除登记，托管 SDK 可在确认后删除实体文件。
_Avoid_: 卸载 SDK、清理系统环境、静默删除文件

**重置工具环境**:
恢复工具自身管理的环境配置、current 入口和托管 PATH 片段；托管 SDK 文件只有在用户明确确认后才会删除。
_Avoid_: 系统还原、清空用户环境、静默删除 SDK

**版本源**:
提供开发工具链版本列表、下载地址和校验信息的远程来源，可以是官方源或镜像源。
_Avoid_: 软件仓库、应用商店、插件市场

**镜像源**:
用于改善下载速度或区域可达性的备用版本源，其安装包仍需要通过校验信息确认完整性。
_Avoid_: 第三方仓库、破解源、非可信下载

**下载校验**:
在线下载完成后确认安装包完整性与可信来源的检查；校验失败的安装包不能解压或登记。
_Avoid_: 下载完成、解压验证、镜像信任

**用户级切换**:
只改变当前 Windows 用户环境中生效的开发工具链版本，不影响其他系统用户或机器级配置。
_Avoid_: 系统级切换、全机器切换、管理员切换

**托管 PATH 片段**:
由工具写入并维护的当前用户 PATH 条目，用于指向工具自己的 current 入口路径。
_Avoid_: 全量 PATH 管理、自动清理用户 PATH、系统 PATH 接管

**PATH 冲突提示**:
工具检测到可能遮蔽当前版本的外部 PATH 条目时，向用户说明冲突来源和手动清理方式，但不自动删除用户配置。
_Avoid_: 自动修复、强制清理、静默重排

**轻量验证**:
切换后只确认 current 入口指向目标 SDK 且关键命令文件存在，不主动运行外部版本命令。
_Avoid_: 命令验证、完整诊断、性能测试

**切换回滚**:
切换失败时自动恢复到切换前的 current 入口，避免开发工具链版本停留在半切换状态。
_Avoid_: 手动恢复、失败停留、调试模式

**手动命令验证**:
由用户主动触发的验证动作，通过运行版本命令确认当前终端解析到的开发工具链版本。
_Avoid_: 自动验证、切换流程、后台扫描

**滚动日志**:
工具按时间和大小分割保存的本地日志，用于记录启动、切换、下载、验证和错误事件。
_Avoid_: 调试输出、命令回显、用户环境快照

**诊断包**:
用户主动导出的排错资料集合，包含脱敏后的日志、配置摘要和诊断结果；不包含完整环境变量、完整 PATH、认证信息或用户目录全量扫描结果。
_Avoid_: 崩溃转储、完整环境变量、隐私数据包

**错误等级**:
工具向用户展示问题严重程度的分类，例如信息、警告、错误和致命错误。
_Avoid_: 日志等级、任务状态、版本状态

**普通权限运行**:
工具默认在当前用户权限下运行，不要求管理员权限，也不主动写入机器级配置。
_Avoid_: 管理员运行、系统级提权、后台服务权限

**版本识别**:
工具通过 SDK 根目录中的关键文件和版本命令确认开发工具链类型与真实版本。
_Avoid_: 目录名猜测、用户手填版本、下载文件名

**GUI 优先**:
第一版以图形界面作为主要交互方式，不要求用户通过命令行完成日常管理和切换操作。
_Avoid_: CLI 优先、命令行工具、终端工作流

**原生前端**:
DevSwitch 的图形界面使用 C#、WinUI 3、XAML 和 MVVM 构建，不使用 Electron、WebView 或 Web 前端栈作为主界面。
_Avoid_: Web 前端、Electron 前端、HTML 界面

**SDK 分类视图**:
按开发工具链类别组织版本列表的界面结构，用户先选择类别再管理该类别下的版本。
_Avoid_: Dashboard、命令面板、混合列表

**版本列表**:
展示同一 SDK 类型下已登记或已托管开发工具链版本的列表。
_Avoid_: 下载列表、任务队列、插件列表

**版本名称**:
用户为某个开发工具链版本设置的显示名称，用于在版本列表中快速识别该版本。
_Avoid_: 真实版本、安装路径、发行版名称

**版本状态**:
某个开发工具链版本在工具中的当前可用性或生效情况，例如使用中、可用、不可用或未验证。
_Avoid_: 下载状态、系统状态、健康报告

**使用中**:
当前 current 入口指向的开发工具链版本。
_Avoid_: 默认版本、系统版本、选中行

**可用**:
关键命令文件存在且最近一次验证未发现该开发工具链版本自身异常的状态。
_Avoid_: 使用中、已安装、已下载

**不可用**:
开发工具链版本的路径不存在、关键命令文件缺失，或 current 入口无法指向它的状态。
_Avoid_: 未安装、禁用、失败任务

**未验证**:
开发工具链版本刚被登记或下载完成，尚未完成命令验证的状态。
_Avoid_: 不可用、待下载、待安装

**数据根目录**:
工具保存配置、下载缓存、托管 SDK、current 入口和日志的用户可写目录。
_Avoid_: C 盘硬编码目录、安装目录、系统目录

**便携数据目录**:
位于工具程序同级或其子目录下的数据根目录，适合免安装或 U 盘携带场景。
_Avoid_: 安装目录写入、系统级数据目录、强制便携模式

**配置迁移**:
工具版本升级后把旧配置转换为当前配置结构的过程，迁移前需要保留可恢复的旧配置副本。
_Avoid_: 重置配置、导入配置、同步配置
